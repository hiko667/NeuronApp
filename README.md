## Uruchomienie

Aby uruchomić ten projekt należy wykonać następujące polecenia, będąc w sklonowanym z repozytorium katalogu

```bash
  python3 -m venv venv
  source ./venv/bin/acticate
  pip install -r requirements.txt
  python3 main.py
```
